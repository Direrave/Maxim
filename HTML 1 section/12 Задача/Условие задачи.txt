Альтернативой class в css служит id. Атрибуты id помещают элементы определённым именем для 
последующего поиска. Если class может назначаться многим элементам, id присваивается одному
определённому элементу. Воспринимай этот идентификатор как номер водительского удостоверения или номер паспорта
