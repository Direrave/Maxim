Рассортируй пациентов.
Рассортируй раненых солдат в три массива.
Первый массив предназначен для мага (mage), которая будет снимать проклятье солдат.
Второй массив - для доктора (doctor), которая будет лечить серьёзно раненых солдат.
Третие массив - для помощника (helper), который найдёт место отдохнуть для легко раненых бойцов.
Стоит напомнить, что добавлять элементы в массив можно следующим образом :
 doctorPatients.push(soldier);
